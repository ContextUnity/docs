## Why ContextCommerce?

ContextCommerce combines Django's robust e-commerce framework with ContextUnity's AI capabilities:

- **Intelligent Product Management** — AI agents automate matching, content generation, and taxonomy
- **Admin Copilot** — Natural language interface for managing products and content
- **Horoshop Sync** — Automated synchronization with Horoshop API
- **Vector Search** — Semantic search across products, brands, and articles

ContextCommerce is built for the ContextUnity ecosystem:

- **Django Integration** — Built on Django Oscar e-commerce framework
- **ContextUnity Native** — Full integration with ContextBrain, ContextRouter, and ContextWorker
- **Production Tested** — Used in real-world e-commerce applications
